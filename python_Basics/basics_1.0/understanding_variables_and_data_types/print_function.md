# How Does the Print Function Work?

Every programming language has a way to output data to the terminal using a built-in method, function, property, or keyword.

In Python, you use the `print()` function to display data in the terminal.

## Hello World with `print()`

One of the first programs people write when learning a language is a **Hello world** program.

In Python, you can do that like this:

```python
print('Hello world!')
```

Output:

```text
Hello world!
```

## Strings in `print()`

In the example above, `'Hello world!'` is a **string**.

A string is a sequence of characters surrounded by single quotes (`'`) or double quotes (`"`).

## Arguments passed to `print()`

In this call:

```python
print('Hello world!')
```

`'Hello world!'` is an **argument** passed to the `print()` function.

## Printing multiple values

You can print multiple values at once by separating them with commas:

```python
print('My favorite colors are', 'blue', 'green', 'red')
```

Output:

```text
My favorite colors are blue green red
```

Python automatically adds a space between items separated by commas.

This makes `print()` useful for combining and displaying multiple pieces of information in one line.
